Politique de Confidentialité

Bienvenue sur le site de Marie Nutrition Coaching. Nous nous engageons à protéger et à respecter votre vie privée. Cette politique de confidentialité explique comment nous collectons, utilisons, stockons et protégeons vos données personnelles lorsque vous utilisez notre site.

1. Nature des Données Collectées

Nous collectons les données suivantes via nos formulaires :

  • Nom
  • Prénom
  • Adresse e-mail
  • Numéro de téléphone
  • Adresse postale
  • Poids
  • Taille

2. Utilisation des Données

Les données collectées sont utilisées uniquement pour :

  • Vous fournir les services de coaching nutritionnel
  • Vous permettre d'accéder à vos informations personnelles
  • Communiquer avec vous en cas de besoin

3. Stockage des Données

Vos données personnelles sont stockées sur une base de données hébergée par Hostinger. Nous nous assurons que nos fournisseurs d'hébergement respectent les normes de sécurité adéquates.

4. Sécurité des Données

Nous mettons en place les mesures de sécurité suivantes pour protéger vos données :

  • Utilisation de mots de passe sécurisés pour l'accès à la base de données
  • Utilisation de requêtes SQL préparées pour prévenir les injections SQL

5. Droits des Utilisateurs

Conformément au RGPD, vous avez le droit de :

  • Accéder à vos données personnelles
  • Rectifier vos données personnelles
  • Supprimer vos données personnelles
  • Limiter le traitement de vos données
  • Opposer au traitement de vos données

Pour exercer ces droits, vous pouvez nous contacter à l'adresse e-mail suivante : marienutrition86@gmail.com

6. Cookies et Technologies de Suivi

Nous n'utilisons pas de cookies ni d'autres technologies de suivi sur notre site.

7. Transferts Internationaux de Données

Nous ne transférons aucune donnée personnelle en dehors de l'Union Européenne.
